## Numbers 

```js
typeof 1 // 'number'

1 + 1 // 2

1 + 2 * 3 // 7 simple maths

NaN !== NaN // true
```

## Strings

```js
typeof 'hello' // string

const name = 'jean';
'hello' + name // 'hellojean' - concatenation
`hello ${name}` // 'hello jean' - template string
```
